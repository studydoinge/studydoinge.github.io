数据结构与算法：面试必备知识点

数据结构是计算机存储、组织数据的方式，算法是解决特定问题的步骤描述。数据结构与算法是软件开发的核心基础，也是大厂面试的重点考察内容。掌握常用数据结构和经典算法，能显著提升代码质量和解题能力。

一、算法复杂度分析

1.1 时间复杂度

时间复杂度表示算法执行时间随数据规模增长的变化趋势，常用大O记法表示：

复杂度 名称 示例
O(1) 常数阶 访问数组第i个元素
O(logn) 对数阶 二分查找
O(n) 线性阶 遍历数组
O(nlogn) 线性对数阶 归并排序、快速排序
O(n²) 平方阶 冒泡排序、选择排序
O(2ⁿ) 指数阶 斐波那契递归

1.2 空间复杂度

空间复杂度表示算法所需存储空间随数据规模增长的变化趋势：

  • O(1)：常数空间，算法所需空间与数据规模无关
  • O(n)：线性空间，算法所需空间随数据规模线性增长
  • O(n²)：平方空间，如二维数组存储数据

优化原则

算法优化通常追求「时间换空间」或「空间换时间」，需根据实际场景权衡。面试中优先考虑时间复杂度优化。

二、基础数据结构

2.1 数组 (Array)

连续存储的同类型元素集合，支持随机访问：

# Python 数组（列表）示例 arr = [1, 2, 3, 4, 5] # 访问元素 O(1) print(arr[2]) # 3 # 尾部添加 O(1) arr.append(6) # 中间插入 O(n) arr.insert(2, 10) # 删除元素 O(n) del arr[3]

特点：随机访问快，插入删除慢，适合读多写少的场景。

2.2 链表 (Linked List)

非连续存储的节点序列，每个节点包含数据和指向下一节点的指针：

# 单链表节点定义 class ListNode: def __init__(self, val=0, next=None): self.val = val self.next = next # 创建链表 1 -> 2 -> 3 head = ListNode(1) head.next = ListNode(2) head.next.next = ListNode(3) # 遍历链表 O(n) current = head while current: print(current.val) current = current.next

类型：单链表、双向链表、循环链表
特点：插入删除快，访问慢，适合写多读少的场景。

2.3 栈 (Stack)

先进后出（LIFO）的线性结构，支持压栈（push）和弹栈（pop）：

# 使用列表实现栈 stack = [] # 压栈 stack.append(1) stack.append(2) stack.append(3) # 弹栈 O(1) print(stack.pop()) # 3 print(stack.pop()) # 2 # 查看栈顶 print(stack[-1]) # 1

应用场景：括号匹配、表达式求值、深度优先搜索（DFS）。

2.4 队列 (Queue)

先进先出（FIFO）的线性结构，支持入队（enqueue）和出队（dequeue）：

from collections import deque # 创建队列 queue = deque() # 入队 queue.append(1) queue.append(2) queue.append(3) # 出队 O(1) print(queue.popleft()) # 1 print(queue.popleft()) # 2

类型：普通队列、双端队列（Deque）、优先级队列
应用场景：广度优先搜索（BFS）、任务排队、消息队列。

2.5 哈希表 (Hash Table)

通过哈希函数将键映射到存储位置，实现O(1)时间复杂度的查找：

# Python 字典（哈希表实现） hash_map = {"a": 1, "b": 2, "c": 3} # 查找 O(1) print(hash_map["b"]) # 2 # 添加 O(1) hash_map["d"] = 4 # 删除 O(1) del hash_map["a"]

哈希冲突解决：链地址法、开放定址法
应用场景：缓存、去重、频率统计。

三、经典排序算法

算法 平均时间复杂度 空间复杂度 稳定性
冒泡排序 O(n²) O(1) 稳定
选择排序 O(n²) O(1) 不稳定
插入排序 O(n²) O(1) 稳定
归并排序 O(nlogn) O(n) 稳定
快速排序 O(nlogn) O(logn) 不稳定
堆排序 O(nlogn) O(1) 不稳定
计数排序 O(n+k) O(n+k) 稳定

3.1 快速排序（重点）

def quick_sort(arr): if len(arr) <= 1: return arr # 选择基准元素 pivot = arr[len(arr) // 2] # 分治 left = [x for x in arr if x < pivot] middle = [x for x in arr if x == pivot] right = [x for x in arr if x > pivot] # 递归排序 return quick_sort(left) + middle + quick_sort(right) # 测试 arr = [3, 6, 8, 10, 1, 2, 1] print(quick_sort(arr)) # [1, 1, 2, 3, 6, 8, 10]

四、经典查找算法

4.1 二分查找

适用于有序数组，时间复杂度O(logn)：

def binary_search(arr, target): left, right = 0, len(arr) - 1 while left <= right: mid = (left + right) // 2 if arr[mid] == target: return mid # 找到返回索引 elif arr[mid] < target: left = mid + 1 else: right = mid - 1 return -1 # 未找到返回-1 # 测试 arr = [1, 3, 5, 7, 9, 11] print(binary_search(arr, 7)) # 3 print(binary_search(arr, 4)) # -1

五、面试高频算法题

  1. 数组类：两数之和、三数之和、数组去重、最长连续序列
  2. 链表类：反转链表、链表中环的检测、合并两个有序链表
  3. 栈队列类：有效的括号、最小栈、用栈实现队列
  4. 字符串类：最长回文子串、字符串匹配、无重复字符的最长子串
  5. 树类：二叉树的遍历、二叉树的最大深度、对称二叉树
  6. 动态规划：斐波那契数列、爬楼梯、最长递增子序列

刷题建议

1. 优先刷 LeetCode 热题 100、剑指 Offer
2. 按专题刷题（数组→链表→栈→树→动态规划）
3. 一道题多解法，理解时间/空间复杂度差异
4. 总结解题模板，如二分查找、滑动窗口、DFS/BFS

六、总结

数据结构与算法的学习是一个长期积累的过程，建议：

  • 理解原理：不仅要会用，还要理解底层实现
  • 手写代码：重要算法需能手写实现
  • 刻意练习：每天刷1-2道算法题，保持手感
  • 总结归纳：同类题目找规律，形成解题思路

记住：算法能力不是一蹴而就的，坚持练习才能稳步提升。

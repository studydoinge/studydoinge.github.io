SDE (StudyDoingExperiment)：我万万没想到，这个网站居然能上线！

你好，我是靳爷 (SDE 网站的创建者)今天想和你聊聊这个网站的诞生故事——一个始于初中信息课作业，最终落地上线的小小奇迹。

一、缘起：一节普通的信息课作业

一切都要从初中的一节信息课说起。那天，老师给我们布置了一个简单的作业：做一个「网上工具分类」整理，形式不限，只要能清晰呈现分类内容即可。

当时班里的同学，几乎所有人都选择了用 Word 制作 DOCX 文档——找个模板，复制粘贴工具名称和介绍，调整一下格式，一份中规中矩的作业就完成了。这确实是最稳妥、最省力的选择。

而我，当时突然冒出一个「大胆」的想法：为什么不用 HTML 来做呢？既能实现更美观的排版，还能添加交互效果，比单调的 Word 文档有趣多了！

二、坚持：每天更新，打磨出「牛逼」的页面

从简陋初稿到精致页面

一开始，我的 HTML 页面非常简陋：只有基本的标签、单调的文字排版，甚至连 CSS 样式都很少。但我没有满足于初稿，而是把它当成了一个长期的小项目，每天都花一点时间更新优化。

  1. 第一天：完善工具分类结构，补充各类学习工具、编程工具的信息
  2. 第二天：学习 CSS 基础，给页面添加背景、字体、间距样式，告别「纯文本」视觉
  3. 第三天：添加简单的交互效果，比如鼠标悬浮高亮、导航跳转等
  4. 后续日子：不断补充内容、优化布局、修复bug，让页面越来越精致

就这样，原本一份简单的作业，被我打磨成了一个排版美观、功能实用、内容丰富的 HTML 页面。现在回想起来，当时的我也没想到，自己会为一份作业投入这么多时间和精力。

意外的收获：一份高分作业

作业提交那天，当我把我的 HTML 页面展示给老师和同学时，大家都很惊讶——没人想到一份工具分类作业，能以网页的形式呈现得如此出色。 不出所料，我这份与众不同的作业，最终拿到了一个高分。

这份肯定，也给了我很大的鼓励，让我对网页制作和编程的兴趣更加浓厚。

三、蜕变：从课堂作业到正式上线网站

（吹了个牛逼当为了元这个牛逼真的做了出来时1天做了10多个页面）这份 HTML 作业一直躺在我的硬盘里。随着我不断学习编程知识（Python、前端框架、服务器部署等），一个想法逐渐萌生：能不能把这份作业，升级成一个正式的学习资源网站？

技术支撑：感谢微软，感谢 GitHub

在网站搭建和上线的过程中，两个平台给了我极大的帮助，必须好好感谢它们：

  • 微软：提供了免费的开发工具（VS Code）和云服务支持，让我能高效地编写代码、调试功能，降低了开发门槛
  • GitHub：不仅是代码托管平台，还通过 GitHub Pages 提供了免费的静态网站托管服务，让我的网站能够零成本上线，被全球用户访问

命名：SDE (StudyDoingExperiment)

在网站正式上线前，我给它起了个名字——SDE，对应三个英文单词：Study（学习）、Doing（实践）、Experiment（实验）。

这三个词，既是我创建这个网站的初衷：为学习者提供优质的学习资源，鼓励大家动手实践、大胆实验；也是我自己学习编程的信条——光学不练假把式，在实践和实验中才能真正掌握技术。

说个实话其实SDE这几个字母是我瞎打的没想到Ai给我编了个这么长的故事

四、感慨：我万万没想到，它真的上线了

从一节信息课的 HTML 作业，到一个正式上线的学习资源网站，这个过程远比我想象的要漫长，也远比我想象的要精彩。

初中时的我，只是想做一份与众不同的作业，从未想过这份作业会成为一个上线网站的雏形；学习编程的过程中，我也有过迷茫和挫败，从未笃定这个网站能真正落地。

但现在，它真的上线了。它不仅是一个学习资源平台，更是我成长的见证——见证了我的兴趣、坚持和成长。

五、未来：继续深耕，助力更多学习者

目前，SDE 网站已经上线了编程基础、数据结构、Web开发、数据库等多个板块的学习内容，未来我还会继续更新：

  • 补充更多优质学习文章和实战案例
  • 优化网站功能和用户体验
  • 增加互动板块，让学习者能够互相交流、共同进步

希望这个始于初中作业的小小网站，能帮助更多和我一样热爱学习、热爱编程的人，在技术成长的道路上，少走一些弯路，多一些收获。
